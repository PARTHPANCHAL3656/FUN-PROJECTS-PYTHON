## üîß Troubleshooting

### "Python is not recognised as a command"
- **Solution**: Reinstall Python and check "Add Python to PATH" during installation
- Or add Python to PATH manually

### "No module named 'requests'"
- **Solution**: Run `pip install requests` in your terminal

### "Permission Denied" error
- **Windows**: Run Command Prompt as Administrator
- **Mac/Linux**: Use `sudo pip install requests`

### Script closes immediately
- Run the script from the terminal/command prompt (not by double-clicking)
- Or add `input("Press Enter to exit...")` at the end of the script

### Rate limit or connection errors
- Check your internet connection
- Some APIs have rate limits (wait a few minutes and try again)
